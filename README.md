# Physical AI Humanoid Robotics Textbook

Welcome to the Physical AI Humanoid Robotics Textbook - an AI-authored technical book with embedded RAG chatbot for answering questions about humanoid robotics.

## About This Book

This book provides comprehensive coverage of humanoid robotics concepts, focusing on practical implementation and real-world applications. Each module builds on fundamental concepts to provide a deep understanding of how to develop and work with humanoid robots.

## Current Modules

### Module 1: The Robotic Nervous System (ROS 2)
- Introduction to ROS 2 for Physical AI
- ROS 2 Communication Model (Nodes, Topics, Services)
- Robot Structure with URDF

## Getting Started

To run this documentation locally:

1. Make sure all dependencies are installed:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open your browser to `http://localhost:3000` to view the documentation.
